We used python3.

When start running the programming, please wait for 30 seconds - 1 mins for the .csv file to display.

To run the program type:
		 python3 fft.py -m CHOICENUMBER

		 where MODELNUMBER:   1 --- image is converted into its FFT
			     	      2 --- denoising
			      	      3 --- compressing
			      	      4 --- plotting the runtime graphs

To modify the range of denoising: change line 58

To modify compressing: change line 77
